# Task : Machine-Learning-Problem-Identification-Framing

Machine Learning Problem Identification and Framing using real-world industry use cases. This project explains how real-world problems can be converted into Machine Learning problems using classification and recommendation systems.

---

## 📌 Project Overview

Social media platforms generate huge amounts of user data every day. This project focuses on using Machine Learning techniques to:

- Detect inappropriate content and cyberbullying
- Improve user safety
- Recommend personalized content to users
- Enhance user engagement and platform experience

---

## 🎯 Objectives

- Detect harmful or abusive content
- Improve online platform safety
- Analyze user behavior and preferences
- Recommend relevant posts and advertisements
- Increase user engagement and satisfaction

---

## 🤖 Type of Machine Learning Problem

This project includes:

### 1. Classification
Used to classify content as:
- Harmful
- Non-harmful

### 2. Recommendation System
Used to recommend:
- Posts
- Videos
- Advertisements
based on user interests and activity.

---

## 📊 Input Features

- Posts and comments
- Images and videos
- Likes, shares, and clicks
- Browsing history
- User profile information

### 🎯 Target Variables

- Harmful / Non-Harmful Content
- Predicted Recommended Content
